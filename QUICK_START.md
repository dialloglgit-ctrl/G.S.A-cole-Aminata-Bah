# ⚡ DÉMARRAGE RAPIDE - 5 Minutes

## 1️⃣ Ouvrir (30 secondes)

```bash
Double-cliquez sur → index.html
```

Ou ouvrez dans votre navigateur :
```
C:\Users\[VotreNom]\OneDrive\Bureau\Linsan\index.html
```

## 2️⃣ Se Connecter (30 secondes)

```
Écran noir avec logo → Code : 1425
```

✅ **ENTREZ** → Appuyez sur Entrée (ou le système détecte auto à 4 chiffres)

---

## 3️⃣ Ajouter Votre Premier Élève (2 minutes)

1. Menu gauche → **👥 Élèves**
2. Cliquez **"+ Ajouter un élève"** (bouton doré)
3. Remplissez :
   - ✅ Prénom : Jean
   - ✅ Nom : Dupont
   - ✅ Classe : 6ème
   - ⭕ (Optionnel) Parent : Saisissez nom
4. Cliquez **"Enregistrer"**

✅ **L'élève apparaît en bas!** Matricule auto-généré

---

## 4️⃣ Ajouter des Notes (1 minute)

1. Menu gauche → **📊 Bulletins**
2. Sélectionnez l'élève dans dropdown
3. Entrez notes (0-20) pour chaque matière
4. ⚡ **La moyenne se calcule auto!**
5. Cliquez **"ENREGISTRER"** (bouton violet)

✅ **C'est sauvegardé!** (localStorage auto)

---

## 5️⃣ Enregistrer un Paiement (1 minute)

1. Menu gauche → **💰 Scolarités**
2. Bouton **"💵 Encaisser"** (haut droit)
3. Remplissez :
   - Élève : Sélectionner
   - Montant : Ex. 500000
   - Motif : Ex. T1 Scolarité
4. **"Confirmer Paiement"**

✅ **Le tableau de bord se MAJ auto!** (Recettes, Solde)

## ❓ Questions Rapides

### "Mes données disparaissent?"
❌ Non! Elles sont sauvegardées automatiquement.

Vérification :
- Ferme l'onglet complètement
- Rouvre le site
- ✅ Tes données sont là!

### "Comment générer un certificat?"
1. Menu → **Certificats**
2. Sélectionne l'élève
3. Clique **"Imprimer"** (Ctrl+P) ou **"Exporter PDF"**

## 🚨 Besoin d'Aide?

| Problème | Solution |
|----------|----------|
| Données perdues | Ferme cache (Ctrl+Shift+Supp) puis F5 |
| Bouton ne répond pas | Esc puis réessaye |
| Code incorrect | Efface & retape (1425) |
| PDF ne télécharge pas | Vérifie pop-ups autorisés |
| Photos grosse erreur | Réduis photo < 2MB |
